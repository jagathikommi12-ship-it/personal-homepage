## How to customize

**Replace the placeholder photo:** drop your real photo in `assets/`, then update the `src` in `index.html` under the About section.

**Replace the placeholder resume:** drop your resume PDF in `assets/` as `resume-placeholder.pdf` (or update the filename in the navbar's Resume button link in `index.html`).

**Replace the GitHub link:** search `index.html` for `<!-- PLACEHOLDER: replace # with your real GitHub URL -->` and swap in your actual GitHub profile/repo link.

**Change colors:** all 4 theme colors are defined as CSS variables at the top of `style.css` under `:root`. Change them there and the whole site updates.

## Tech
Plain HTML5 + CSS3, Google Fonts (Space Grotesk + Inter), no build step required — just open `index.html` in a browser.
